# 金融类自定义view

## 介绍
> 本项目会对金融交易软件中存在的各种View进行模仿绘制，提供详细的实现思路，收集整理相关算法、文档以及专业资料。

## 存在的意义
* 可能有同学会说，有很成熟的第三方图形图啊，干嘛还去自己实现？
* 知道如何编写绘制，更方便的扩展第三库图形库。
* 了解实现原理，如果第三方实现不了，可以从容的自己撸一个。
* 金融交易中各种算法、各种指标很多，但是网上相关资料很少。希望可以给金融相关软件从业者提供一点点帮助~
* 知其然，知其所以然。
